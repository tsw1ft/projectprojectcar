# Project: Project Car

Purpose:
This website will allow users to share their project car(s) and update fellow enthusiasts on the state of their project. It will allow users to find fellow enthusiasts and socialise with them on the app, possibly allowing them to organise in person meets or drives. This project will personally teach me how to program in Node.js as well as improve my software development skills.

Functions:
1. Discover Page
   a. Scroll through other projects
2. Following Page
   a. Show recent project updates from cars you're following
3. Profile Page
   a. Person info
   b. Car collection
4. Car Page
   a. Car picture
   b. Recent updates
   c. To-do list
5. Intro Page
   a. Sign in/up
   b. Explain purpose of site
